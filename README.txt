Description:

This project is a small and simple example for getting and validating the responses from pet store swagger api
for various scenarios on different end points 

End points covered for Pet store API:

1. Find a pet by ID
2. Update the pets name by pet id
3. Delete the pet name by id

Implementation details:

Used Rest assured using Java 8 precisely, Cucumber, Log 4j and JUnit

Usage:

Run using Maven command - mvn test

Report:
test-output.html can be found under root folder and can be viewed in browser
